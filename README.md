# Blog Pessoal - Frontend React

Frontend do Blog Pessoal desenvolvido com React + TypeScript + Vite.

## Tecnologias

- React 19
- TypeScript
- Vite

## Como Executar

```bash
# Instalar dependências
npm install

# Executar em desenvolvimento
npm run dev

# Build para produção
npm run build
```

## Estrutura

```
src/
├── pages/
│   └── home/
│       └── Home.tsx    # Página inicial
├── App.tsx             # Componente raiz
└── main.tsx            # Entrada da aplicação
```
